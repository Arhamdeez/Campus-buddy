# CampusBuddy

CampusBuddy is a context-aware student assistant web application designed to help university students manage their academic and campus life. It features chat, announcements, lost & found, anonymous feedback/confessions, gamification, and AI-powered tools.

## Tech Stack
- Frontend: React (TypeScript, Vite)
- Backend: Node.js (Express, TypeScript)
- Database: Firebase Firestore, PostgreSQL
- AI/NLP: OpenAI API

## Monorepo Structure
- `client/` – React frontend
- `server/` – Node.js backend
- `shared/` – Shared types/interfaces
- `docs/` – Documentation
